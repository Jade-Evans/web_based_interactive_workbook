TOPIC:

Default CSS Styles

COURSE DETAILS:

The Odin Project (JS FS), Intermediate CSS

5 Key Facts:

  1. To ensure legibility, responsive and accessible styling by default, browsers have their own default stylesheet.Egs. include the varying font-sizes of h1, h2 etc. or the automatic styling of a button element.
  2. Default CSS can cause complications, as there can be subtle but irritating differences between browser defaults which cause inconsistencies. Developers also have to spend time undoing these presets to achieve their desired aesthetic.
  3. Developers have 3 main approaches to overcome these issues, as outlined below:
