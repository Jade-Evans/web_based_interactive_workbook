The Odin Project - JavaScript Full Stack - Intermediate CSS

5 Key Facts:

  1. To ensure legibility, responsive and accessible styling by default, browsers ship with their own default stylesheet. Examples include heading font sizes and the automatic styling of button elements.
  2. Default CSS varies subtly between browsers, which can cause layout and styling inconsistencies. Developers often spend time undoing these presets to achieve consistent design.
  3. Developers use three main approaches to overcome these issues:
